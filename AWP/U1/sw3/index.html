¿Qué hace este Service Worker?

Este Service Worker guarda en cache los archivos de esta página para que funcione sin conexión a internet.

Archivos que se almacenan:

  • Esta página (index.html)
  • El archivo JavaScript (app.js)
  • El Service Worker (sw.js)
  • El icono de la página
Presiona el botón para activar el Service Worker
